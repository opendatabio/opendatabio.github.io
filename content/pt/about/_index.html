Software de código aberto para facilitar o estudo de biodiversidade!

{{< blocks/link-down color="warning" >}} {{< /blocks/cover >}} {{< blocks/section color="ligth" >}}

O projeto OpenDataBio nasceu da necessidade prática de organizar dados usados em estudos de biodiversidade na Amazônia e Mata Atlântica:

  • Organizar, curar e interligar os dados coletados pelo Projeto Dinâmica Biológica de Fragmentos Florestais (PDBFF) do INPA desde o início dos anos 80, bem como nas parcelas do ForestGeo no Brasil (Ilha do Cardoso e Manaus)
  • Integrar dados no nível de indivíduo (organismo) para estudos de taxonomia integrativa, partindo da premissa de que as definições de espécies são instáveis (na Amazônia) ​​e requerem o compartilhamento dos dados brutos ​​para definir e descrever as espécies, e alimentar repositórios para IA
  • Ter flexibilidade no armazenamento de atributos, mas ao mesmo tempo ser capaz de padronizar adequadamente os dados e de interligalos minimizando redundâncias
  • Ser capaz de curar e gerenciar dados de forma contínua, acompanhando o histórico de alterações, e mais facilmente compartilhar dados, diretamente em contextos locais (servidor na Instituição, laboratório ou grupo de pesquisa) e também para facilitar a exportação de versões estáticas dos dados para os repositórios públicos
